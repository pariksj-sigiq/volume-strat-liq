Read This First

What this run is showing

This screen backtests a long liquidity-sweep setup on the selected stock's continuous front-month futures series, while NIFTY50 stays as the benchmark. Each triggered setup uses the prior red candle low as the fixed stop, applies your entry offset from that stop anchor, and recalculates target distance from the visible 1:R control.

How to use it

  1. Pick a stock, date window, reward ratio, entry offset, and walk-forward limit.
  2. Run analysis, then scan the overview chart for where sweep signals appeared versus NIFTY50.
  3. Click a trade card or ledger row to replay the candle-level setup with entry, stop, target, and exit bands.

What the columns mean

Signal
date the sweep setup triggered
Exit
date target, stop, or timeout resolved
Expiry
front-month futures contract used
Outcome
target, stop, timeout, or ambiguous result
Entry / Stop / Target
trade levels calculated from the stop anchor and 1:R ratio
Return
percentage move from entry to realized exit
Net P&L / Lot
rupee result after applying the contract lot size
Held
number of futures bars held
63D Spread
symbol's 63-day return minus NIFTY50's 63-day return at signal time

Important context

The strategy instrument is stock futures, not cash equity. Coverage depends on what Upstox returned locally for each contract, so a requested 5-year window can still have less than 5 years of actual futures history for some symbols.
